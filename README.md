# 纸飞机工程师 Demo

题目 5「纸飞机工程师 —— 从折纸到超临界翼型」的静态网页 Demo。

## 玩法

打开 `index.html` 后，先选择折纸工位里的折法，也可以手动调整机翼面积、重心位置、折叠角度、翼尖形状、投掷角度和力度，然后选择远航、滞空或投送关卡，点击「投掷」。

页面会实时显示飞行距离、滞空时间、速度、迎角、升阻比和姿态稳定性，并在落地后给出工程反馈。

## 展示重点

- 用纸飞机参数对应基础空气动力学概念：升力、阻力、重心、翼尖稳定性。
- 引入侧风、乱流、地效和失速区间，让飞行结果更像工程调参。
- 用变色轨迹、速度矢量、投掷点和地面刻度表现不同设计方案的飞行结果。
- 支持 2D/3D 视角切换，3D 视角会显示机翼面、上反角、滚转姿态和透视跑道；拖拽可旋转视角，Shift 拖拽可平移，滚轮缩放，双击复位。
- 支持 3D 折纸预览动画，纸张会沿中线、机翼折线和机头折线逐步翻折成型。
- 3D 纸飞机增加了投影阴影、纸张厚度、折痕线和高光，飞行姿态更接近实体纸模。
- 用任务化目标把科普内容变成可玩的设计沙盒。
- 支持数值输入、键盘微调和本机保存/载入预设，便于反复对比方案。
- 折法会自动映射到机翼面积、重心、折叠角、翼尖和投掷建议。
- 每次试飞后生成复盘报告：最大迎角、最低升阻比、失速时长、落地冲击和改进建议。
- 每个关卡会保存本机最佳成绩、星级和本次徽章。
- A/B 试验台可以保存两个方案，用同一套物理模型快速对比距离、滞空、迎角、升阻比和冲击。
- 风洞模式会固定纸飞机并显示气流线、升力/阻力/重力箭头和失速红区。
- 首次打开会显示 4 步新手引导，也可以随时点击画布右上方的“引导”重新查看。
